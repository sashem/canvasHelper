¿Dónde estoy?

Canvashelper es una herramienta experimental que facilita la creación de modelos de negocio empleando la lógica de "canvas bussiness model generation".

En una primera instancia, este sitio debiera servirte para generar modelos de forma sencilla y visualmente atractiva, sin embargo se plantea que a futuro sirva como una plataforma pedagógica orientada a mejorar el uso del canvas en todo tipo de empresarios.

Te invitamos a participar de esta versión de prueba enviando un correo a mimesis17@gmail.com para crear una cuenta de usuario.

Esta aplicación está hecha para funcionar correctamente en Chrome y Firefox. No funciona en Internet Explorer y no está probada en otros exploradores.
